# project-server-2019-deployment-plan
Enterprise deployment project plan for Project Server 2019 developed in Microsoft Project, including WBS development, resource allocation, budgeting, scheduling, dependencies, and milestone tracking.

## Project Summary

This project was developed using Microsoft Project and includes planning, scheduling, budgeting, resource management, dependency tracking, and milestone management for a Project Server 2019 deployment.

## Major Work Packages

1. Plan for Project Server 2019 Deployment
2. Prepare for Deployment
3. Install SharePoint Server 2019
4. Install and Configure Project Server 2019
5. Create the Project Web App Site

## Project Metrics

- Project Duration: 96 days
- Total Budget: $70,860
- Multiple employee resources assigned
- Hardware resources included:
  - Dell Server
  - Laptop
- Milestones established for each major phase

## Skills Demonstrated

- Work Breakdown Structure (WBS)
- Project Scheduling
- Resource Allocation
- Dependency Management
- Milestone Tracking
- Enterprise Deployment Planning
- Microsoft Project

## What I Learned

This project helped me develop practical project management skills by planning a multi-phase enterprise software deployment, managing resources, sequencing tasks, and building a complete project schedule in Microsoft Project.
